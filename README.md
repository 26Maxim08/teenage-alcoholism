## Что делает проект
По анкете студента модель оценивает 
**вероятность того, что он относится к группе повышенного потребления алкоголя**, 
и относит его к классу «группа риска» или «низкий риск». Это задача
**бинарной классификации**.

## Данные
Датасет **Student Alcohol Consumption** (UCI Machine Learning Repository),
файл `data/student-mat.csv` — 395 студентов.

Целевая переменная строится из двух колонок опроса о потреблении алкоголя:
`Dalc` (будни, шкала 1–5) и `Walc` (выходные, 1–5), взвешенных по числу дней:

```
weekly = Dalc * 5 + Walc * 2
класс «группа риска» (1), если weekly >= 14   (примерно верхняя треть выборки)
```

Колонки `Dalc` и `Walc` **исключены из признаков** — иначе была бы утечка.
Баланс классов: риск 30.9% / норма 69.1%.

## Отбор признаков (упрощение модели)
Из 30 доступных признаков отобраны 17: ядро самых информативных плюс несколько интуитивных полей (возраст, пропуски и т.п.). Полная
модель на 30 признаках заметно переобучалась (CV ROC-AUC ≈ 0.72); сокращённая
даёт сопоставимое качество при гораздо более простой анкете.

## Модель и обучение (`train_model.py`)
   | Модель | ROC-AUC (CV) |
   |---|---|
   | **Логистическая регрессия** | **0.719 ± 0.057** |
   | Случайный лес | 0.696 ± 0.055 |
   | Градиентный бустинг | 0.637 ± 0.054 |

## Веб-интерфейс (`app.py`)
Flask-приложение. Форма строится автоматически из схемы признаков, сохранённой
вместе с моделью. Есть кнопка «Случайный студент», шкала риска с крупным
показателем вероятности и подвижный порог решения.

## Быстрый старт

pip install -r requirements.txt
